"Социальная сеть"
Университетный проект.

Нужно реализовать 3 следующие функции:
	1) void like(char *name, char *user)
		Пользователь user страничку "name".
	2) void unlike(char *page, char *user)
		Пользователь user дизлайкнул страничку "page".
	3) char *getuser(char *name, int k)
		Нужно как можно быстрее найти k-того пользователь в списке всех пользователей, которые лайкнули страничку "name".

Решение:
	Хэш-таблица для хранения данных о страничке. Данные представляют собой название странички и АВЛ-дерево лайкнувших пользователей.

Файл draw.c нужен для рисования дерева в функции randomtest(), где тестируются random-ные input-ы.
