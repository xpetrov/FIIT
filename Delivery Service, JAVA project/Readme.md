Университетский проект. "Служба доставки"

Заданием было придумать любой проект на логистику, чтобы объекты между собой как-то взаимодействовали.

Решение:
	Служба доставки. Есть 2 варианты пользования: как заказчик и как админ.
	Заказчик указывает, что он хочет отправить, размеры посылки, вариант доставки и откуда-куда. Ему возвращается номер для отслеживания посылки.
	Админ может отследить информацию о каждой посылке. Отдает команду единицам транспорта "в путь" (типо "Поехали!").

!!! Для работы нужно скопировать все txt-файлы в папку с проектом.
